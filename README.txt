Team Task Manager - Full Stack Web Application

Project Overview:
A Django-based full-stack web application where users can create projects, assign tasks, and track progress with role-based access control.

Features:
- User Authentication (Signup/Login/Logout)
- Role-Based Access (Admin/Member)
- Project Management
- Task Creation and Assignment
- Dashboard with Task Statistics
- Overdue Task Highlighting
- Live Deployment on Railway

Tech Stack:
- Backend: Django
- Frontend: HTML, Bootstrap
- Database: SQLite
- Deployment: Railway
- Version Control: GitHub

Admin Features:
- Create Projects
- Create Tasks
- Assign Tasks

Member Features:
- View Assigned Tasks
- Track Task Status
